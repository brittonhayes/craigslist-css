craigslist-css

A Tailwind CSS Theme Capturing the Craigslist Aesthetic

Welcome to craigslist-css Documentation

This page demonstrates all components and utilities available in the craigslist-css Tailwind theme. Each example includes the code you need to use it in your own project.

No fancy gradients, no smooth animations, no rounded corners. Just pure, functional web design.

Installation

1. Configure npm for GitHub Packages

Create or edit .npmrc in your project:

@brittonhayes:registry=https://npm.pkg.github.com

2. Install the package

npm install @brittonhayes/craigslist-css

3. Include in your HTML

<link href="./dist/output.css" rel="stylesheet">

Buttons

Search Bar
