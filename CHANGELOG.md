# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-27

### Added
- Initial release of craigslist-css
- Complete Tailwind CSS theme with Craigslist aesthetic
- Color palette matching classic Craigslist design
- Pre-built components:
  - Buttons (`cl-button`)
  - Inputs and selects (`cl-input`, `cl-select`)
  - Cards (`cl-card`)
  - Links (`cl-link`)
  - Lists (`cl-list`)
  - Tables (`cl-table`)
  - Sections (`cl-section`)
  - Alerts (`cl-alert`, `cl-alert-error`, `cl-alert-success`)
  - Header and footer (`cl-header`, `cl-footer`)
- Typography utilities
- Demo page showcasing all components
- Comprehensive documentation

### Design Principles
- Zero border radius
- No box shadows
- Classic blue links (#0000ff)
- Purple visited links (#551a8b)
- Dense, minimal spacing
- Utilitarian aesthetic
